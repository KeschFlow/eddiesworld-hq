Datenschutzerklärung

1. Datenschutz auf einen Blick

Diese Webseite wird über GitHub Pages gehostet. Wir erfassen nur Daten, die technisch notwendig sind oder die du uns aktiv durch die Teilnahme am Spiel (Unaufhaltsam) oder dem Archiv-Voting (KEEP-THIS) übermittelst.

2. Hosting und Infrastruktur

Unsere Website liegt auf Servern von GitHub Inc. (USA). Beim Aufruf der Seite werden Logfiles (IP-Adresse, Zeitstempel) durch den Hoster verarbeitet.

Für interaktive Funktionen (Leaderboard, Archiv-Counter) nutzen wir Firebase (Google Cloud Platform). Die Datenverarbeitung erfolgt in der Region eur3 (Europa).

3. Datenerfassung durch Interaktion

Unaufhaltsam Leaderboard: Wenn du deinen Score speicherst, werden dein gewählter Name, Score, Zeit und eine anonyme Firebase-Auth-ID gespeichert. Dies dient der Darstellung im Leaderboard.

RAW LOGS Counter: Beim Klicken auf "KEEP THIS" wird ein Zähler in Firestore erhöht, um die Beliebtheit von Inhalten zu messen.

4. YouTube & Externe Links

Videos werden im erweiterten Datenschutzmodus über youtube-nocookie.com eingebunden. Ein Datenaustausch mit YouTube findet erst beim Start des Videos statt. Externe Links zu TikTok, Instagram, Reddit oder Merch-Shops führen dich auf die Plattformen der jeweiligen Anbieter; dort gelten deren Datenschutzbestimmungen.

5. Deine Rechte

Du hast das Recht auf Auskunft, Berichtigung und Löschung deiner gespeicherten Daten. Kontaktiere uns dazu einfach per E-Mail.
